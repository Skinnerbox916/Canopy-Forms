Usage Instructions

  1. Replace YOUR_SITE_API_KEY with your actual site API key
  2. Update data-can-o-form values to match your form slugs
  3. Customize theme tokens as needed
  4. Ensure the origin matches your site's configured domain

Available Theme Tokens

  • fontFamily - CSS font-family string
  • fontUrl - URL to font stylesheet (Google Fonts, etc.)
  • text - Text color (hex)
  • background - Form background color (hex)
  • primary - Primary accent color (hex)
  • border - Input border color (hex)
  • radius - Border radius in pixels (number)
  • density - Spacing density: "compact" | "normal" | "comfortable"
